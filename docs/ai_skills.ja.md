# エージェントスキル (Beta)

エージェントスキルは、ポータブルで自己完結型の指示セットであり、AI コーディングアシスタント（Antigravity、Claude Code、Cursor など）が Godot AdMob エディタープラグインの API、規約、パターンをより高い精度で理解できるように支援します。

各プラットフォームの情報で AI のシステムプロンプトを過負荷にするのではなく、エージェントスキルはプログレッシブディスクロージャー（段階的開示）を活用します。これにより、AI アシスタントは特定のタスクを実行するために必要な指示とリソースだけを読み込むことができます。

---

## スキルをインストールする

エディタを使用して、事前構築済みの AdMob エージェントスキルを Godot プロジェクトに直接インストールできます。

1. Godot プロジェクトを開きます。
2. エディタの上部メニューで **Project > Tools > AdMob Manager** をクリックします。
3. **AI Copilot** タブを選択します。
4. **Install AI Skills to Project** をクリックします。

これにより、プロジェクトのルートディレクトリに `.skills/` フォルダが自動的に作成され、利用可能なスキルがその中に配置されます。

!!! note
    ドットで始まるファイルやフォルダ（`.skills` など）は、Godot の FileSystem ドックではデフォルトで非表示になっていますが、外部 IDE や AI コーディングアシスタントからは完全に表示されます。

### スキルを更新する
AI アシスタントが最新の機能やベストプラクティスにアクセスできるように、スキルを最新の状態に保ってください。更新するには、Godot エディターで AdMob Manager を開き、再度 **Install AI Skills to Project** をクリックして、フォルダを最新のファイルで上書きします。

---

## スキルを呼び出す
スキルがプロジェクトリポジトリにインストールされると、AI アシスタントは自動的にそれらをインデックスして検出します。プロンプトで明示的に `@` 記号を使用する必要はありません。AdMob について質問すると、アシスタントは自動的にスキルのコンテキストを参照します。

アシスタントに特定のスキルを手動で強制的に使用させたい場合は、`@` に続けて名前を入力して参照できます。例：
`@godot-admob-banner`

---

## 利用可能なスキルを表示する

次の表は、Godot AdMob エディタープラグインの機能で利用可能なすべてのスキルについて説明しています：

| スキル | 説明 |
| :--- | :--- |
| `godot-admob-get-started` | Google Mobile Ads SDK の初期化、リクエスト構成の設定、User Messaging Platform (UMP) 同意フローの構成を支援します。 |
| `godot-admob-banner` | バナー広告（`AdView`）の実装、広告サイズの選択、折りたたみ式バナーの設定を支援します。 |
| `godot-admob-interstitial` | 全画面インタースティシャル広告の実装、広告の読み込み、閉じるコールバックの設定を支援します。 |
| `godot-admob-rewarded` | リワード広告およびリワードインタースティシャル広告の実装、ユーザー獲得報酬イベントのリッスン、広告参照のクリーンアップを支援します。 |
| `godot-admob-app-open` | アプリの起動時やアプリの遷移時に表示される App Open 広告の実装を支援します。 |
| `godot-admob-native-overlay` | カスタマイズされた小または中サイズの Control ノードテンプレートを使用した Native Overlay 広告の実装を支援します。 |
| `godot-admob-migrate` | 異なるプラグインバージョン間（例: v4.x から v5.x）でのプロジェクトの移行を支援します。 |
| `godot-admob-mediation` | サードパーティのメディエーションネットワーク（AppLovin、Meta Audience Network、Unity Ads など）の構成、統合、検証を支援します。 |
| `godot-admob-troubleshoot` | 広告読み込みエラーの診断、エラーコードの処理、Ad Inspector の起動、統合の問題の解決を支援します。 |
| `godot-admob-privacy` | ユーザー同意（UMP フロー）、GDPR 準拠、COPPA（子供向け取り扱い）、CCPA、リクエストターゲティングオプションの構成を支援します。 |

お気に入りの AI アシスタントにプロンプトを送ることで、Godot AdMob エディタープラグインのエージェントスキルを実装できます。エージェントスキルは、AI アシスタントが次の例のようなタスクを実行するのをガイドできます：

* *Godot AdMob エディタープラグインの初期化を手伝って。*
* *私のゲームにバナー広告を追加して。*
* *リワード広告を実装して。*
* *プロジェクトを v4 から v5 に移行して。*
* *GDPR と COPPA のプライバシーコンプライアンスを構成して。*
* *広告が読み込まれない理由をトラブルシューティングして。*
