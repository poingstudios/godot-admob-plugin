# Solucionar problemas de configurações de privacidade

Você pode verificar suas configurações de privacidade e consentimento diretamente no inspecionador de anúncios, incluindo:

- Status do consentimento (TCF)
- Configurações da plataforma de mensagens do usuário (UMP)
- Tag para menores de idade de consentimento
- Tag para tratamento direcionado a crianças
