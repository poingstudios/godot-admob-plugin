# Vpon

Vpon メディエーションアダプターを使用すると、AdMob メディエーションを使用して Vpon 広告を Godot ゲームに統合できます。

## サポートされている統合

| 広告フォーマット | Bidding | Waterfall |
| :--- | :--- | :--- |
| **バナー** | ❌ | |
| **インタースティシャル** | ❌ | |
| **リワード** | ❌ | |

## 統合手順

### Android

#### 1. プロジェクト設定で Vpon を有効にする
- Godot プロジェクト設定の `admob/android/mediation/vpon` で **Vpon** を有効にします。

#### 2. AdMob コンソールでメディエーションを設定する
AdMob コンソールで Vpon をメディエーションパートナーとして設定します。
1. **AdMob アカウント**にログインします。
2. **メディエーション**に移動し、**メディエーション グループ**を作成または編集します。
3. 広告ネットワークウォーターフォールで、Vpon を**カスタム イベント**として追加します。
4. **クラス名 (Class Name)**: `com.vpadn.mediation.VpadnAdapter` と入力します。
5. **パラメータ**: Vpon ライセンスキーを入力します。

---

### iOS

iOS では Vpon がカスタムイベントを介して統合されるため、開発者はエクスポート時に Vpon SDK と AdMob アダプターフレームワークファイルを Xcode ビルド設定に追加する必要があります。

#### 1. AdMob コンソールでメディエーションを設定する
1. **AdMob アカウント**にログインします。
2. **メディエーション**に移動し、**メディエーション グループ**を作成または編集します。
3. 広告ネットワークウォーターフォールで、Vpon を**カスタム イベント**として追加します。
4. **クラス名 (Class Name)**: `VponAdMobAdapter` と入力します。
5. **パラメータ**: Vpon ライセンスキー / 広告ユニット ID を入力します。
