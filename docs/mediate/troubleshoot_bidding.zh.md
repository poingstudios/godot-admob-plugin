# 解决出价问题

出价 (Bidding) 允许广告源实时竞价你的广告库存。本页提供了常见出价问题的排查步骤。

## 常见出价问题

### 广告未加载

如果使用出价时广告未加载：

1. **检查适配器版本**：确保你使用的是最新版本的出价适配器
2. **验证初始化**：检查 SDK 和出价适配器是否已正确初始化
3. **检查广告单元配置**：验证你的广告单元是否在 AdMob UI 中配置为出价

### 填充率低

如果在使用出价时遇到填充率低的问题：

1. **检查广告源状态**：验证广告源是否处于活动状态且配置正确
2. **检查底价设置**：检查你的底价是否设置得过高
3. **地理覆盖范围**：某些广告源的地理覆盖范围有限

### 延迟问题

如果出价导致延迟：

1. **检查网络状况**：较差的网络状况会增加出价延迟
2. **监控适配器延迟**：使用初始化状态来检查适配器延迟
3. **优化超时设置**：如果可用，调整超时设置

## 调试工具

### 广告检查器 (Ad Inspector)

使用广告检查器来调试出价问题：

1. 在应用中启用广告检查器
2. 在广告加载时启动广告检查器
3. 检查出价状态和响应信息

### 日志

启用日志以查看详细的出价信息：

=== "GDScript"

    ```gdscript
    # 启用调试日志
    MobileAds.set_debug(true)
    ```

=== "C#"

    ```csharp
    // 启用调试日志
    MobileAds.SetDebug(true);
    ```
