# 音量コントロール

AdMob 動画広告の音量を制御することで、アプリのオーディオ設定と統合できます。音量コントロールは Android と iOS の両方で動作します。

相対音量（0.0 〜 1.0）を設定するには `set_app_volume()` を、広告音声をミュートまたはミュート解除するには `set_app_muted()` を使用します。

# 前提条件

- [はじめに](index.ja.md) ガイドを完了していること
- Godot v4.2 以上を使用していること

# 使い方

=== "GDScript"

    ```gdscript
    # 広告の音量を設定 (0.0 = 消音, 1.0 = 最大音量)
    MobileAds.set_app_volume(0.5)
    
    # 広告をミュートまたはミュート解除
    MobileAds.set_app_muted(true)
    ```

=== "C#"

    ```csharp
    // 広告の音量を設定 (0.0 = 消音, 1.0 = 最大音量)
    MobileAds.SetAppVolume(0.5f);
    
    // 広告をミュートまたはミュート解除
    MobileAds.SetAppMuted(true);
    ```

# API リファレンス

**set_app_volume(volume: float)**

広告再生の相対音量を設定します。値は 0.0（消音）から 1.0（現在のデバイス音量）の範囲にクランプされます。

**set_app_muted(muted: bool)**

アプリの音声がミュートされたときに AdMob SDK に通知します。`true` の場合、広告の音声がミュートされます。

# 重要な注意点

- これらの設定は動画広告（インタースティシャル、リワード、リワードインタースティシャル、アプリ起動時）に適用されます。バナー広告やネイティブ広告はカスタム音量コントロールをサポートしていない場合があります。
- 音量設定は、広告の読み込み前後を問わず、いつでも変更できます。
- 一貫したユーザー体験を提供するために、これらの呼び出しをアプリ内のゲーム音量スライダーや設定画面と同期させてください。
