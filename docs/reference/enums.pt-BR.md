# Enums

Esta página lista os enums públicos usados pelo Godot AdMob Plugin.

## AdPosition

Define a posição do banner de anúncio na tela.

=== "GDScript"
    ```gdscript
    enum AdPosition {
        TOP = 0,
        BOTTOM = 1,
        LEFT = 2,
        RIGHT = 3,
        TOP_LEFT = 4,
        TOP_RIGHT = 5,
        BOTTOM_LEFT = 6,
        BOTTOM_RIGHT = 7,
        CENTER = 8
    }
    ```

=== "C#"
    ```csharp
    public enum AdPosition
    {
        Top = 0,
        Bottom = 1,
        Left = 2,
        Right = 3,
        TopLeft = 4,
        TopRight = 5,
        BottomLeft = 6,
        BottomRight = 7,
        Center = 8
    }
    ```
