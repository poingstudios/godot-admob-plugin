# APIリファレンス

!!! note "Godot 3 (v1) ドキュメント"
    このページは **Godot 3.x** 用です。**Godot 4.2+** については、[安定版ドキュメント](https://poingstudios.github.io/godot-admob-plugin/stable/) をご覧ください。

このセクションには、Godot AdMobプラグインが提供するクラス、メソッド、プロパティ、列挙型の技術リファレンスが含まれています。

---

## 利用可能なセクション

*   **[MobileAds API](api.md)**: `MobileAds`シングルトンが提供するメソッドとシグナルの完全なリスト。
*   **[列挙型](enums.ja.md)**: グローバル設定、ターゲティング、レイアウト列挙型のリスト。
