# API リファレンス

このセクションには、Godot AdMob プラグインが提供するクラス、メソッド、プロパティ、および列挙型（Enums）の技術リファレンスが含まれています。

サイドバーからクラスを選択すると、**GDScript** と **C#** の両方の実装詳細を表示できます。

## 利用可能なセクション

*   **[クラス](classes/AdView.ja.md)**: `AdView` などの Godot AdMob API クラスの技術仕様。
*   **[列挙型 (Enums)](enums.ja.md)**: グローバル構成、ターゲティング、およびレイアウトに関する列挙型のリスト。
