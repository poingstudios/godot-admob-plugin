# 枚举和配置

本页列出了Godot 3中Godot AdMob插件使用的公共枚举和配置。

---

## INITIALIZATION_STATUS

定义Google Mobile Ads SDK的就绪状态。

*   `NOT_READY` = `0`
*   `READY` = `1`

---

## POSITION

定义广告横幅在屏幕上的位置。

*   `BOTTOM` = `0`
*   `TOP` = `1`

---

## BANNER_SIZE

在AdMob编辑器面板设置中配置的横幅尺寸：

*   `"BANNER"`（标准横幅：320x50）
*   `"MEDIUM_RECTANGLE"`（中矩形：300x250）
*   `"FULL_BANNER"`（全横幅：468x60）
*   `"LEADERBOARD"`（排行榜：728x90）
*   `"ADAPTIVE"`（锚定自适应横幅）
*   `"SMART_BANNER"`（智能横幅：已弃用）
