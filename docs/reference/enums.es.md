# Enums y Configuraciones

!!! note "Documentación Godot 3 (v1)"
    Esta página es para **Godot 3.x**. Para **Godot 4.2+**, consulte la [documentación estable](https://poingstudios.github.io/godot-admob-plugin/stable/).

Esta página enumera los enums públicos y configuraciones utilizados por el Complemento AdMob de Godot en Godot 3.

---

## INITIALIZATION_STATUS

Define el estado de preparación del SDK de Google Mobile Ads.

- `NOT_READY` = `0`
- `READY` = `1`

---

## POSITION

Define la posición del anuncio banner en la pantalla.

- `BOTTOM` = `0`
- `TOP` = `1`

---

## BANNER_SIZE

El tamaño del banner configurado en los ajustes del panel del Editor de AdMob:

- `"BANNER"` (Banner estándar: 320x50)
- `"MEDIUM_RECTANGLE"` (Rectángulo mediano: 300x250)
- `"FULL_BANNER"` (Banner completo: 468x60)
- `"LEADERBOARD"` (Leaderboard: 728x90)
- `"ADAPTIVE"` (Banner adaptivo anclado)
- `"SMART_BANNER"` (Banner inteligente: obsoleto)
