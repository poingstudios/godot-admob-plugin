# AdValue.PrecisionType

広告インプレッションについて報告された金額の精度を指定します。[`AdValue`](../classes/AdValue.md) クラス内で定義されています。

## 値

=== "GDScript"
    ```gdscript
    # Access via AdValue.PrecisionType
    enum PrecisionType {
        UNKNOWN = 0,
        ESTIMATED = 1,
        PUBLISHER_PROVIDED = 2,
        PRECISE = 3
    }
    ```

=== "C#"
    ```csharp
    public enum PrecisionType
    {
        Unknown = 0,
        Estimated = 1,
        PublisherProvided = 2,
        Precise = 3
    }
    ```
