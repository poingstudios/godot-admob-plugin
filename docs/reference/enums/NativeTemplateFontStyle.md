# NativeTemplateFontStyle

Defines the font weight styles for native overlay text fields. Used by [`NativeTemplateTextStyle`](../classes/NativeTemplateTextStyle.md).

## Values

=== "GDScript"
    ```gdscript
    # Access via NativeTemplateFontStyle.Values
    enum Values {
        NORMAL = 0,
        BOLD = 1,
        ITALIC = 2,
        MONOSPACE = 3
    }
    ```

=== "C#"
    ```csharp
    public enum NativeTemplateFontStyle
    {
        Normal = 0,
        Bold = 1,
        Italic = 2,
        Monospace = 3
    }
    ```
