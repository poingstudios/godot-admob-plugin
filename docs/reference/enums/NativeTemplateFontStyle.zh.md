# NativeTemplateFontStyle

定义原生叠加文本字段的字重样式。由 [`NativeTemplateTextStyle`](../classes/NativeTemplateTextStyle.md) 使用。

## 值

=== "GDScript"
    ```gdscript
    # Access via NativeTemplateFontStyle.Values
    enum Values {
        NORMAL = 0,
        BOLD = 1,
        ITALIC = 2,
        MONOSPACE = 3
    }
    ```

=== "C#"
    ```csharp
    public enum NativeTemplateFontStyle
    {
        Normal = 0,
        Bold = 1,
        Italic = 2,
        Monospace = 3
    }
    ```
