# AdChoicesPlacement

定义原生广告上 AdChoices 叠加图标的角落位置。由 [`NativeAdOptions`](../classes/NativeAdOptions.md) 使用。

## 值

=== "GDScript"
    ```gdscript
    # Access via AdChoicesPlacement.Values
    enum Values {
        TOP_RIGHT = 0,
        TOP_LEFT = 1,
        BOTTOM_RIGHT = 2,
        BOTTOM_LEFT = 3
    }
    ```

=== "C#"
    ```csharp
    public enum AdChoicesPlacement
    {
        TopRight = 0,
        TopLeft = 1,
        BottomRight = 2,
        BottomLeft = 3
    }
    ```
