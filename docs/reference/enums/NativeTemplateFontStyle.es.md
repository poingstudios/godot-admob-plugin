# NativeTemplateFontStyle

Define los estilos de peso de fuente para los campos de texto de superposición nativa. Usado por [`NativeTemplateTextStyle`](../classes/NativeTemplateTextStyle.md).

## Valores

=== "GDScript"
    ```gdscript
    # Access via NativeTemplateFontStyle.Values
    enum Values {
        NORMAL = 0,
        BOLD = 1,
        ITALIC = 2,
        MONOSPACE = 3
    }
    ```

=== "C#"
    ```csharp
    public enum NativeTemplateFontStyle
    {
        Normal = 0,
        Bold = 1,
        Italic = 2,
        Monospace = 3
    }
    ```
