# NativeMediaAspectRatio

Configures the aspect ratio preference for media content in native ads.

## Values

=== "GDScript"
    ```gdscript
    # Access via NativeMediaAspectRatio.Values
    enum Values {
        UNKNOWN = 0,
        ANY = 1,
        LANDSCAPE = 2,
        PORTRAIT = 3,
        SQUARE = 4
    }
    ```

=== "C#"
    ```csharp
    public enum NativeMediaAspectRatio
    {
        Unknown = 0,
        Any = 1,
        Landscape = 2,
        Portrait = 3,
        Square = 4
    }
    ```
