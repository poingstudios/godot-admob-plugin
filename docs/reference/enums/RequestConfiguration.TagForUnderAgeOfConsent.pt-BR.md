# RequestConfiguration.TagForUnderAgeOfConsent

Indica se as solicitações de anúncio devem ser tratadas como direcionadas a usuários menores de idade para conformidade com o GDPR. Definido dentro da classe [`RequestConfiguration`](../classes/RequestConfiguration.md).

## Valores

        UNSPECIFIED = -1,

        TRUE = 1
    }
