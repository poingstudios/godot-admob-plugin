# AdPosition

定义横幅和原生叠加广告的屏幕对齐方式。由 [`AdView`](../classes/AdView.md)、[`NativeOverlayAd`](../classes/NativeOverlayAd.md) 和 [`AdPosition`](../classes/AdPosition.md) 类使用。

## 值

=== "GDScript"
    ```gdscript
    # Access via AdPosition.Values
    enum Values {
        TOP = 0,
        BOTTOM = 1,
        LEFT = 2,
        RIGHT = 3,
        TOP_LEFT = 4,
        TOP_RIGHT = 5,
        BOTTOM_LEFT = 6,
        BOTTOM_RIGHT = 7,
        CENTER = 8,
        CUSTOM = -1
    }
    ```

