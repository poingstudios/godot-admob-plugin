# AdChoicesPlacement

ネイティブ広告上の AdChoices オーバーレイアイコンの角の配置を定義します。[`NativeAdOptions`](../classes/NativeAdOptions.md) で使用されます。

## 値

=== "GDScript"
    ```gdscript
    # Access via AdChoicesPlacement.Values
    enum Values {
        TOP_RIGHT = 0,
        TOP_LEFT = 1,
        BOTTOM_RIGHT = 2,
        BOTTOM_LEFT = 3
    }
    ```

=== "C#"
    ```csharp
    public enum AdChoicesPlacement
    {
        TopRight = 0,
        TopLeft = 1,
        BottomRight = 2,
        BottomLeft = 3
    }
    ```
