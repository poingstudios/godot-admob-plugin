# AdChoicesPlacement

Define la colocación en la esquina del ícono de superposición de AdChoices en anuncios nativos. Usado por [`NativeAdOptions`](../classes/NativeAdOptions.md).

## Valores

=== "GDScript"
    ```gdscript
    # Access via AdChoicesPlacement.Values
    enum Values {
        TOP_RIGHT = 0,
        TOP_LEFT = 1,
        BOTTOM_RIGHT = 2,
        BOTTOM_LEFT = 3
    }
    ```

=== "C#"
    ```csharp
    public enum AdChoicesPlacement
    {
        TopRight = 0,
        TopLeft = 1,
        BottomRight = 2,
        BottomLeft = 3
    }
    ```
