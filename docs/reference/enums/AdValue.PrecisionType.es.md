# AdValue.PrecisionType

Especifica la precisión del valor monetario informado para una impresión de anuncio. Definido dentro de la clase [`AdValue`](../classes/AdValue.md).

## Valores

=== "GDScript"
    ```gdscript
    # Access via AdValue.PrecisionType
    enum PrecisionType {
        UNKNOWN = 0,
        ESTIMATED = 1,
        PUBLISHER_PROVIDED = 2,
        PRECISE = 3
    }
    ```

