# AdValue.PrecisionType

Especifica a precisão do valor monetário relatado para uma impressão de anúncio. Definido dentro da classe [`AdValue`](../classes/AdValue.md).

## Valores

=== "GDScript"
    ```gdscript
    # Access via AdValue.PrecisionType
    enum PrecisionType {
        UNKNOWN = 0,
        ESTIMATED = 1,
        PUBLISHER_PROVIDED = 2,
        PRECISE = 3
    }
    ```

=== "C#"
    ```csharp
    public enum PrecisionType
    {
        Unknown = 0,
        Estimated = 1,
        PublisherProvided = 2,
        Precise = 3
    }
    ```
