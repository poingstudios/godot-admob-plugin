# RequestConfiguration.TagForUnderAgeOfConsent

広告リクエストを GDPR 準拠のために同意年齢未満のユーザーを対象として扱うかどうかを示します。[`RequestConfiguration`](../classes/RequestConfiguration.md) クラス内で定義されています。

## 値

=== "GDScript"
    ```gdscript
    # Access via RequestConfiguration.TagForUnderAgeOfConsent
    enum TagForUnderAgeOfConsent {
        UNSPECIFIED = -1,
        FALSE = 0,
        TRUE = 1
    }
    ```

=== "C#"
    ```csharp
    public enum TagForUnderAgeOfConsent
    {
        Unspecified = -1,
        False = 0,
        True = 1
    }
    ```
