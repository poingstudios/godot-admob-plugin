# AdPosition

バナーおよびネイティブオーバーレイ広告の画面配置を定義します。[`AdView`](../classes/AdView.md)、[`NativeOverlayAd`](../classes/NativeOverlayAd.md)、および [`AdPosition`](../classes/AdPosition.md) クラスで使用されます。

## 値

=== "GDScript"
    ```gdscript
    # Access via AdPosition.Values
    enum Values {
        TOP = 0,
        BOTTOM = 1,
        LEFT = 2,
        RIGHT = 3,
        TOP_LEFT = 4,
        TOP_RIGHT = 5,
        BOTTOM_LEFT = 6,
        BOTTOM_RIGHT = 7,
        CENTER = 8,
        CUSTOM = -1
    }
    ```

=== "C#"
    ```csharp
    public enum Values
    {
        Top = 0,
        Bottom = 1,
        Left = 2,
        Right = 3,
        TopLeft = 4,
        TopRight = 5,
        BottomLeft = 6,
        BottomRight = 7,
        Center = 8,
        Custom = -1
    }
    ```
