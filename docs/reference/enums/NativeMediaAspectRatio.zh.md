# NativeMediaAspectRatio

配置原生广告中媒体内容的宽高比偏好。由 [`NativeAdOptions`](../classes/NativeAdOptions.md) 使用。

## 值

=== "GDScript"
    ```gdscript
    # Access via NativeMediaAspectRatio.Values
    enum Values {
        UNKNOWN = 0,
        ANY = 1,
        LANDSCAPE = 2,
        PORTRAIT = 3,
        SQUARE = 4
    }
    ```

=== "C#"
    ```csharp
    public enum NativeMediaAspectRatio
    {
        Unknown = 0,
        Any = 1,
        Landscape = 2,
        Portrait = 3,
        Square = 4
    }
    ```
