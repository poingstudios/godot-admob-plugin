# RequestConfiguration.TagForUnderAgeOfConsent

Indica si las solicitudes de anuncios deben tratarse como dirigidas a usuarios menores de edad para el cumplimiento del GDPR. Definido dentro de la clase [`RequestConfiguration`](../classes/RequestConfiguration.md).
