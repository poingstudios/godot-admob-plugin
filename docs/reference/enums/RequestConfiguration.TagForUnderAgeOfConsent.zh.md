# RequestConfiguration.TagForUnderAgeOfConsent

指示广告请求是否应被视为面向未达到同意年龄的用户以符合 GDPR 规定。定义在 [`RequestConfiguration`](../classes/RequestConfiguration.md) 类内部。

        TRUE = 1
    }
