# NativeMediaAspectRatio

Configura a preferência de proporção de aspecto para conteúdo de mídia em anúncios nativos. Usado por [`NativeAdOptions`](../classes/NativeAdOptions.md).

## Valores

=== "GDScript"
    ```gdscript
    # Access via NativeMediaAspectRatio.Values
    enum Values {
        UNKNOWN = 0,
        ANY = 1,
        LANDSCAPE = 2,
        PORTRAIT = 3,
        SQUARE = 4
    }
    ```

=== "C#"
    ```csharp
    public enum NativeMediaAspectRatio
    {
        Unknown = 0,
        Any = 1,
        Landscape = 2,
        Portrait = 3,
        Square = 4
    }
    ```
